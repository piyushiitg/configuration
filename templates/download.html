The Knowlus Installation simplifies the process of installing Knowlus.

Prerequisites

Install the following software before installing the Python Helper library.

  • Python >= 2.7.3 - Most Linux installations come installed with Python; if you are using a Windows system, see your local documentation to install Python. Check out the www.python.org/getit site also to download and install Python.

  • Installation

    • Add Repository in sources.list
      • sudo vim /etc/apt/sources.list

      or

      sudo gedit /etc/apt/sources.list
    • Add the following line to the list and save it.
      • deb http://10.60.20.136/repo/ /
      deb http://10.60.20.136/repopack/ /
    • Now update the system.
      • sudo apt-get update
    • Install Package
      • sudo apt-get install knowluspackage
    • During installation while creating knowlus database it will ask you some details. Give username & password as "knowlus" & give some email id if it asks.
    • If the installation is done correctly the six services it runs in the end will work fine. The final output will be something like this:

      Starting Services

      * Restarting knowlus rpc server for IO intensive tasks krpycd [ OK ]

      * Restarting knowlus rpc server for IO intensive tasks krpycd_incalld [ OK ]

      * Restarting outgoing call handler outcalld [ OK ]

      * Restarting incoming call handler incalld [ OK ]

      * Restarting preprocessor daemon preprocessd [ OK ]

      * Restarting postprocessor daemon postprocessd [ OK ]
